Introduction aux Bases de Données

Les bases de données (BDD) sont des systèmes organisés de stockage d'informations...

  • Tables : Les bases de données sont constituées de tables. Chaque table est similaire à un tableau.
  • Enregistrements : À l'intérieur des tables, les données sont stockées sous forme d'enregistrements, qui correspondent à des lignes. Chaque enregistrement contient des informations spécifiques.
  • Clé Primaire (PK) : Il est crucial de définir une clé primaire dans chaque table, généralement sous la forme d'une colonne ID. La clé primaire identifie de manière unique chaque enregistrement dans la table.

Opérations CRUD

Les opérations CRUD sont fondamentales dans la manipulation des données au sein des bases de données...

  • Create (Créer) : Permet d'ajouter de nouveaux enregistrements à une table.
  • Read (Lire) : Facilite la consultation et la récupération d'informations depuis la base de données.
  • Update (Mettre à jour) : Permet de modifier des enregistrements existants.
  • Delete (Supprimer) : Utilisé pour effacer des enregistrements de la base de données.

Architecture 3 Tiers

L'architecture 3 tiers est un modèle commun dans le développement de logiciels basés sur des bases de données...

  • Client - Serveur - Base de Données : Cette architecture divise l'application en trois composants distincts, améliorant la modularité et la maintenance.

Système de Gestion de Bases de Données Relationnelles (SGBDr)

Les SGBDr sont des logiciels conçus pour gérer les bases de données relationnelles. Le langage principal utilisé est le SQL...

  • SQL (Structured Query Language) : Langage permettant de manipuler les données dans une BDD.
  • Documentation : La documentation officielle du SQL est disponible sur sql.sh.

Principaux SGBDr

Il existe plusieurs systèmes de gestion de bases de données relationnelles, chacun ayant ses caractéristiques spécifiques...

  • MySQL
  • PostgreSQL
  • MariaDB
  • SQLite
  • Oracle
