Cours sur Git et GitHub

Introduction

Git est un système de contrôle de version qui permet de suivre l'évolution du code source d'un projet. GitHub est une plateforme de collaboration basée sur Git.

Commandes Git Principales

            
                git clone <url_du_depot>        # Cloner un dépôt
                git checkout -b nom-de-la-branche     # Créer et basculer vers une nouvelle branche
                git checkout nom-de-la-branche        # Basculer vers une branche existante
                git status                            # Vérifier l'état des fichiers modifiés
                git add nom-du-fichier                # Ajouter un fichier au suivi
                git add .                             # Ajouter tous les fichiers modifiés au suivi
                git commit -m "Votre message"         # Valider les modifications avec un message
                git push origin nom-de-la-branche     # Pousser vers une branche distante
                git pull origin nom-de-la-branche     # Récupérer les modifications depuis une branche distante
                git log                               # Voir l'historique des commits

Créer une Pull Request sur GitHub

Après avoir effectué des modifications sur une nouvelle branche :

  1. Ouvrez votre dépôt GitHub dans un navigateur.
  2. Cliquez sur "Compare & pull request".

Apprenez et maîtrisez Git et GitHub pour une gestion efficace de vos projets de développement !
