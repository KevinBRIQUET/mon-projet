Introduction à Git et GitHub

Git est un système de contrôle de version très populaire, et GitHub est une plateforme d'hébergement de projets Git. Ce cours vous introduira aux concepts fondamentaux de Git et vous montrera comment utiliser GitHub pour collaborer sur des projets.

Leçon 1 : Installation de Git

Avant de commencer à utiliser Git, vous devez l'installer sur votre machine. Suivez les instructions appropriées pour votre système d'exploitation.

sudo apt-get install git brew install git

Leçon 2 : Configuration de Git

Configurez Git avec vos informations personnelles, telles que votre nom d'utilisateur et votre adresse e-mail.

git config --global user.name "Votre Nom" git config --global user.email "votre@email.com"

Leçon 3 : Création d'un dépôt Git

Apprenez à créer un nouveau dépôt Git pour suivre les modifications de votre projet.

git init

Leçon 4 : Commandes Git de base

Découvrez les commandes Git de base pour commencer à versionner votre code.

  • git add: Ajouter des fichiers à l'index
  • git commit: Créer un nouveau commit
  • git status: Vérifier l'état des fichiers
  • git log: Afficher l'historique des commits

Leçon 5 : Branches Git

Comprenez le concept de branches et apprenez à créer, fusionner et supprimer des branches.

  • git branch: Afficher la liste des branches
  • git checkout: Changer de branche
  • git merge: Fusionner des branches
  • git branch -d: Supprimer une branche

Leçon 6 : Collaboration avec GitHub

Explorez les fonctionnalités de collaboration de GitHub, y compris les forks, les pull requests et la gestion des conflits.

  • git clone: Cloner un dépôt distant
  • git pull: Mettre à jour votre dépôt local
  • git push: Pousser vos modifications vers un dépôt distant
  • pull request: Proposer des changements à un projet
