Cours sur GitHub

Introduction à Git et GitHub

Git est un système de contrôle de version très populaire, et GitHub est une plateforme d'hébergement de projets Git. Ce cours vous introduira aux concepts fondamentaux de Git et vous montrera comment utiliser GitHub pour collaborer sur des projets.

Leçon 1 : Installation de Git

Avant de commencer à utiliser Git, vous devez l'installer sur votre machine. Suivez les instructions appropriées pour votre système d'exploitation.

sudo apt-get install git brew install git

Leçon 2 : Configuration de Git

Configurez Git avec vos informations personnelles, telles que votre nom d'utilisateur et votre adresse e-mail.

git config --global user.name "Votre Nom" git config --global user.email "votre@email.com"

Leçon 3 : Création d'un dépôt Git

Apprenez à créer un nouveau dépôt Git pour suivre les modifications de votre projet.

git init
